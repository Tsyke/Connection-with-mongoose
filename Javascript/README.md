# FR

## Connection avec MongoDB

### Première étape
Clonez ce repo

```
git clone https://github.com/tsyke/Connection-with-mongoose
```

### Seconde étape
Installez tout les module

```
cd ./Javascript
npm i
```

### Dernière étape
Run app and ⭐ :)

```
node ./index.js
```
